# Tools

A collection of small utilities I build to support my workflow as a frontend platform / design system engineer.  
These tools help me track time, automate repetitive tasks, and improve productivity while maintaining high-quality UI components.

---

## Current Tools

### Time Tracker

**A minimal timer to understand how my time is distributed across development, support, and leadership work while running a design system team.**

- **Live Demo:** [GitHub Pages Link](https://craighagemeier.github.io/tools/time-tracker/)  
- **Purpose:** Quickly track how much time I spend on different types of work, helping me optimize my workflow and report on team progress.  
- **Tech Stack:** HTML, CSS, JavaScript  
- **Usage:**  
  Open the [live demo](https://craighagemeier.github.io/tools/time-tracker/) in your browser to start using the timer.

---

## Why These Tools Exist

I build small utilities like the Time Tracker to streamline daily work for myself. Instead of relying on complex third-party tools, these lightweight scripts provide exactly what's needed without unnecessary overhead.

---

## Technical Decisions

- **Vanilla JavaScript:** Keeps the tool lightweight and easy to deploy. No build step required.  
- **HTML/CSS simplicity:** Ensures it works cross-browser and can be hosted directly on GitHub Pages.  
- **Folder structure:** Each tool lives in its own folder with an optional demo and README for clarity.
